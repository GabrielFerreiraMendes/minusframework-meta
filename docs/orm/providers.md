# Providers

O MinusORM suporta 7 bancos de dados via FireDAC.

## Configuração Rápida

### SQLite

```pascal
TConexaoFactory.Criar
  .Driver('SQLite')
  .Database(':memory:')
  .Conectar;
```

### Firebird

```pascal
TConexaoFactory.Criar
  .Driver('Firebird')
  .Database('C:\dados.fdb')
  .Usuario('SYSDBA')
  .Senha('masterkey')
  .Servidor('localhost')
  .Porta(3050)
  .Conectar;
```

### PostgreSQL

TConexaoFactory.Criar

## Pool de Conexões

```pascal
TConexaoFactory.Criar
  .Driver('PostgreSQL')
  .Pool(5, 20)  // mínimo 5, máximo 20 conexões
  .Database('meudb')
  .Conectar;
```

## Mapeamento de Tipos por Provider

| Provider | Booleano | Auto Incremento |
|----------|----------|-----------------|
| SQLite | `INTEGER` (0/1) | `INTEGER PRIMARY KEY AUTOINCREMENT` |
| Firebird | `SMALLINT` (0/1) | `INTEGER GENERATED BY DEFAULT AS IDENTITY` |
| PostgreSQL | `BOOLEAN` | `SERIAL` / `BIGSERIAL` |
| MySQL | `TINYINT(1)` | `INT AUTO_INCREMENT` |
| MariaDB | `TINYINT(1)` | `INT AUTO_INCREMENT` |
| MSSQL | `BIT` | `INT IDENTITY(1,1)` |
| Oracle | `SMALLINT` (0/1) | `NUMBER GENERATED BY DEFAULT AS IDENTITY` |
